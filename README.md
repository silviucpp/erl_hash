## Overview

A collection of hash algorithms that are not available in the standard Erlang libraries.

### Supported Hash Algorithms

- `murmur3_x64_128`
- `murmur2`
- `fnv1_32`
- `fnv1a_32`
- `fnv1_64`
- `fnv1a_64`

## Build

To compile the project, run:

rebar3 compile
